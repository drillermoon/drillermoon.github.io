Research

My research interests lie in the robustness and generalization of deep neural networks in a variety of domains including computer vision, natural language processing, and reinforcement learning. More specifically, I am interested in

  • Adversarial Attacks & Robustness
  • Offline Reinforcement Learning
  • Generalization in Reinforcement Learning.

Publications
Rethinking Value Function Learning for Generalization in Reinforcement Learning
Seungyong Moon, JunYeong Lee, Hyun Oh Song
Neural Information Processing Systems (NeurIPS), 2022
code / bibtex

The value network trained on multiple training environments is more likely to overfit to the training environments and requires sufficiently regularized. We propose a new RL algorithm that updates the value network less frequently with more data for improving generalization.
Query-Efficient and Scalable Black-Box Adversarial Attacks on Discrete Sequential Data via Bayesian Optimization
Deokjae Lee, Seungyong Moon, Junhyeok Lee, Hyun Oh Song
International Conference on Machine Learning (ICML), 2022
code / bibtex

Crafting an adversarial example against language models is challenging due to its discrete nature and dynamic input size. We develop a query-efficient black-bax adversarial attack against various language models from LSTM to BERT via discrete Bayesian optimization.
Preemptive Image Robustification for Protecting Users against Man-in-the-Middle Adversarial Attacks
Seungyong Moon*, Gaon An*, Hyun Oh Song
AAAI Conference on Artificial Intelligence (AAAI), 2022
code / bibtex

Leveraging an intriguing property of deep neural networks that they have robust points near in-distribution datapoints, we propose a new defense framework that preemptively modifies images ahead of adversarial attacks, which can be applied to a realistic scenario.
Uncertainty-Based Offline Reinforcement Learning with Diversified Q-Ensemble
Gaon An*, Seungyong Moon*, Jang-Hyun Kim, Hyun Oh Song
Neural Information Processing Systems (NeurIPS), 2021
code / bibtex

The Q-function ensemble technique developed for addressing overestimation bias in online RL can be also applied to offline RL with Q-function gradient diversification. We propose an offline RL algorithm that does not require behavior cloning or explicit Q-value penalization.
Parsimonious Black-Box Adversarial Attacks via Efficient Combinatorial Optimization
Seungyong Moon*, Gaon An*, Hyun Oh Song
International Conference on Machine Learning (ICML), 2019   (Long Talk, 159/3424=4.6%)
code / bibtex

We develop a query-effecient black-box adversarial attack against deep neural networks based on the local search algorithm for non-monotone submodular function maximization, which does not require gradient estimation and becomes free of hyperparameters to tune.
